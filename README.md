# Doctrines Slack Bot

Use this bot in slack to get a bible verse via a reference.

## Installation

- Get a Bibles.org API key.
- Host this node.js on a server.
- Create a slack command (e.g. `doctrine`) and point the request URL to the hosted server URL.

## Usage

> /doctrine [reference]

## Example:

> /doctrine john 1:1
